DESC

    DESC guides analysts in determining the optimal dimensionality through various novel visualization techniques that are specifically designed for the exploration of the dimensionality space, such as hull heatmap, and dimensionality animation. Hull heatmap provides an overview of gene expression over multiple dimensionalities for multiple cell markers. Each rectangle of the hull heatmap includes a convex hull representing a cluster of the target cells in the given dimensionality. By using cell filtering, analysts can update the hull heatmap by interactively modifying the parameters defining the hulls based on their expertise. To gather more information about cells, analysts can use dimensionality animation. All the t-SNE plots used in DESC were generated by the RunTSNE function of Seurat with default parameters. In the following notes, we explain the design rationale of the DESC and the method used to determine the optimal dimensionality using the proposed system in detail.

More Information on DESC

1. Hull Heatmap

    Hull heatmap shows an overview of significant cells for multiple dimensionalities; therefore users can examine the cells without the requirement of analyzing hundreds of projection plots.

    A large hull indicates that target cells are spread over a wide, while a small hull indicates a close gathering.

2. Marker Expression Plot

    The marker expression plot is a dimensionality reduction plot that is colored based on the expression level of a cell marker.

3. Cluster Plot

    The cluster plot is a dimensionality reduction plot that is colored according to the clustering result of the input Seurat object.

4. Dimensionality Animation

    Dimensionality animation shows the difference between different marker expression plots (for different dimensionalities) by smoothly interpolating between two plots over time.

5. Cell Filtering

    DESC provides intuitive interactive cell filtering functions using the lasso selection, which allows users to progressively update the hull heatmap until they obtain the most satisfactory heatmap and identify the optimal dimensionality.
